npm(全稱Node Package Manager)

是軟體套件管理系統(node預設、js編寫而成)
npm install gulp -g
-g 代表安裝在全域。其所生成的 package.json 會加入到 "devDependencies"，
沒加 -g 會加入到 "dependencies"，而所下載之套件會放入一新生成之資料夾 node_modules，
在 git push 時只用將 package.json 上傳，下載的人使用 npm 時自動會依內容下載所記載之套件，
package-lock.json 會鎖定版本，避免版號前有加 ^ 號時會自動更新，如 ^0.0.1 指下載比 0.0.1 更高之版本。
