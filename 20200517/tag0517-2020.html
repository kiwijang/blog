1.1 變與不變

資訊的時代，『速度』很重要！英打又比中打重要。

所謂的專業資訊技術: 想要應徵 IT 公司的話，有個會 Linux 作業系統，懂得架設 NAT (未來會介紹，就是類似無線基地台的轉址功能) 那就已經是了不起的專業技術。 現在 2019 年呢？這些技術都是基本技能。

雖然某些 邏輯概念、作業系統 與 網路相關基礎觀念、美學設計理念 等或許是萬年不變的基礎， 但在『應用』端，就不可能不變動！因此，要記得隨時更新你腦袋的資訊。

1.2 基礎概念：數字表示法

我們的電腦只認識 0 與 1，記錄的資料也是只能記錄 0 與 1 而已，所以電腦常用的資料是二進位的。

IP 位址、檔案容量計算、下載時間規劃等等，都與這些數字表示方式有關。

  • 2 進位 (binary system)

    最早的電腦只會知道 0 跟 1 (其實就是有沒有通電)
    目前的網路傳輸資料的速度單位，就是以 bit 為主。

  • 資料存放的應用，使用 byte 位元組

    雖然 bit 是電腦的基本資料存放單位，但是 bit 實在是太小了。
    後來使用了 8 個 bit 來作為一些資料存放的最小單位，這就被稱為位元組 (byte)， 也因為使用到 8 個 bit 來存放，所以兩者間的關係就是：
    1 byte = 8 bits
    也就是說，無論你資料是多小，一定會佔用一整個 byte 在做儲存。
